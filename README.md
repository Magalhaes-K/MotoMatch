# Sistema de Comparação de Motos em Java

# Objetivo

Projeto acadêmico desenvolvido para estudos de Programação Orientada a Objetos utilizando Java.

# Descrição

Este projeto foi desenvolvido em Java com o objetivo de comparar características técnicas de motocicletas através de um sistema interativo em console.

O usuário pode selecionar duas motos cadastradas e visualizar suas informações, além de realizar comparações entre diferentes atributos, como cilindrada, preço e peso.

O projeto foi criado com foco na aplicação de conceitos fundamentais de Programação Orientada a Objetos (POO), incluindo classes, objetos, encapsulamento, construtores, métodos e coleções.

---

## Funcionalidades

* Exibição das motos cadastradas.
* Seleção de duas motos para análise.
* Visualização da ficha técnica das motos selecionadas.
* Comparação de cilindrada.
* Comparação de preço.
* Comparação de peso.
* Menu interativo em console.

---

## Tecnologias Utilizadas

* Java
* IntelliJ IDEA / Eclipse (compatível)
* Programação Orientada a Objetos (POO)
* Collections Framework (ArrayList)

---

### Classe Moto

Responsável por representar uma motocicleta através dos atributos:

* Modelo
* Marca
* Tipo
* Cilindrada
* Preço
* Peso

Também contém métodos para:

* Exibir ficha técnica
* Comparar cilindrada
* Comparar preço
* Comparar peso

### Classe Main

Responsável pela interação com o usuário, gerenciamento do menu e execução das funcionalidades do sistema.
